# Lessons-runtime — plateforme

Lieu d'accumulation des leçons opérationnelles tirées du travail quotidien sur ce repo.

## Format

```markdown
### YYYY-MM-DD — Titre court à l'impératif

**Contexte** : ce qui s'est passé (1-3 phrases)
**Leçon** : ce qu'on doit retenir (rule, why, how to apply)
**Tag(s)** : `php-symfony`, `gitnexus`, `mailchimp`, etc.
```

## Promotion

- Lesson généralisable au-delà du projet (autres repos Renaissance) → `/socle-promote-learning` pour proposer en PR sur le repo `socle`.
- Lesson strictement locale (spécifique au domaine) → reste ici.
